fetchAttendance

Fetches attendance for the week contained in the date or if fetchFullMonth is true fetches month instead of single week. May return empty list if fetching week and specified date's day component is Sunday.


Parameters

session: ISession - session tokens

registerId: number - register id

date: Date - reference date

fetchFullMonth: boolean - whether to fetch whole month or just a single week


Returns

Promise<IAttendance>
