IAttendance.IClasses

Contains attendance info for single classes.

There are following attendance types:

  • holidays - 0
  • excused - 1
  • late - 2
  • absence - 3
  • exempt - 4
  • cancelled - 5
  • unknown (?) - 6
  • present - 7
  • trip - 8
  • exempt/present - 9


Fields

classesDate: Date - date

classesTimeSpan: string - the start and end time of the lesson

classesIndex: number - index of classes starting from 1

classesTopic: string - classes topic

type: number - attendance type, see above for explanation

wasEdited: boolean - was this attendance entry edited?

teacher: string - teacher name that conducted the classes

subject: string - subject
