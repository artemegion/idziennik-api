IGrades.IHistory

History of a grade entry.


Fields

issuedDate: number - date when the grade was issued

countsForGpa: boolean - do this grade counts for gpa?

category: string - grade category

color: string - color in which the grade should be displayed

grade: string - string representation of the grade

semester: number - semester this grade was issued for

weight: number - grade weight

valueForGpa: number - grade value (grade * weight)
